<h1>📄 Resgen – Quick Resume Generator</h1>

<p><strong>Resgen</strong> is a fast and simple web application designed to help you create clean, text-based resumes in minutes. It provides custom content blocks and pre-written, best-suited texts to make resume creation faster and easier—perfect for quick applications or last-minute submissions.</p>

<p>Whether you’re building your first resume or updating an existing one, Resgen makes the process effortless with customizable sections and instant formatting.</p>

<hr>

<h2>✨ Features</h2>
<ul>
  <li><strong>⚡ Quick Resume Creation</strong> – Build a professional resume in minutes with predefined text suggestions.</li>
  <li><strong>🧩 Custom Blocks</strong> – Add, remove, and rearrange sections as you see fit.</li>
  <li><strong>🖋️ Pre-Written Texts</strong> – Use curated, best-suited phrases to make your resume stand out instantly.</li>
  <li><strong>💻 Instant Preview</strong> – See your resume update in real time as you make changes.</li>
</ul>
